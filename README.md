# immich-go-sdk

Auto-generated Go SDK for the [Immich](https://immich.app) API.

Generated using [openapi-generator](https://openapi-generator.tech/docs/generators/go/) against the
[Immich OpenAPI spec](https://github.com/immich-app/immich/blob/main/open-api/immich-openapi-specs.json).

## Status

| File | Value |
|------|-------|
| Tracked Immich version | See [`.immich-version`](.immich-version) |
| Spec SHA-256 | See [`.spec-hash`](.spec-hash) |

The SDK is regenerated automatically every day via GitHub Actions. A new commit and tag are pushed
only when the upstream spec or release version changes.

## Usage

```go
import "github.com/harrisonhjones/immich-go-sdk/sdk"
```

The generated code lives in the `sdk/` directory. Refer to the
[openapi-generator Go client docs](https://openapi-generator.tech/docs/generators/go/) for
configuration and usage patterns.

## Manual regeneration

Trigger the workflow manually from the **Actions** tab → **Generate SDK** → **Run workflow**.
